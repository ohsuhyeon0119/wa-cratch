## 태스크

### Task 1 — mock data 중앙화 (3분)
- 파일:
  - `frontend/src/mock/projects.ts`
  - `frontend/src/mock/users.ts`
  - `frontend/src/mock/activity.ts`
- 구현:
  - `Project` 타입: `{ id, title, author, authorId, emoji, likes, views, published, createdAt, description, tags }`
  - `User` 타입: `{ id, username, nickname, avatar, projectCount, followers }`
  - `Activity` 타입: `{ type, icon, text, time }`
  - 각각 10개/3개/4개의 mock 배열 export
  - Dashboard와 Explore가 동일한 Project 배열을 참조하도록 통일
- 검증: TypeScript 타입 오류 없음 (`yarn build` 통과)

### Task 2 — Toast 공통 컴포넌트 (4분)
- 파일:
  - `frontend/src/components/Toast/Toast.tsx`
  - `frontend/src/components/Toast/Toast.module.css`
  - `frontend/src/hooks/useToast.ts`
- 구현:
  - `useToast()` 훅: `{ show, message, type }` 상태 + `showToast(message, type)` 함수
  - `<Toast />` 컴포넌트: 하단 고정, 2.5초 뒤 자동 소멸, fade-in/out 애니메이션
  - type: `'success' | 'info' | 'error'`
- 검증: Editor 저장 버튼 클릭 시 토스트 노출 확인

### Task 3 — ShareModal 공통 컴포넌트 (4분)
- 파일:
  - `frontend/src/components/ShareModal/ShareModal.tsx`
  - `frontend/src/components/ShareModal/ShareModal.module.css`
- 구현:
  - props: `{ isOpen, projectId, projectTitle, onClose }`
  - URL 표시 + "링크 복사" 버튼 클릭 시 `navigator.clipboard.writeText()` 호출
  - 복사 성공 후 버튼 텍스트 "✅ 복사됨!" 으로 1.5초 변경
  - ESC 키 / 배경 클릭 시 닫힘
  - 오버레이 + 모달 다이얼로그 구조
- 검증: Editor/Play 페이지에서 공유 버튼 클릭 시 모달 노출

### Task 4 — DashboardPage mock data 연결 + 인터랙션 (3분)
- 파일: `frontend/src/pages/DashboardPage/DashboardPage.tsx`
- 구현:
  - 하드코딩된 `PROJECTS`, `ACTIVITY` 배열을 `src/mock/`에서 import
  - "💾 저장" 버튼 없음 (Dashboard는 프로젝트 목록 페이지) → 불필요
  - 사이드바 "🚪 로그아웃" 클릭 → `navigate('/')` + 토스트 "로그아웃 됐어요"
- 검증: `npx playwright test e2e/dashboard.spec.ts`

### Task 5 — ExplorePage 검색/정렬 실동작 (4분)
- 파일: `frontend/src/pages/ExplorePage/ExplorePage.tsx`
- 구현:
  - `src/mock/projects.ts` import
  - `searchQuery` state 추가, 검색창 onChange → 필터링
  - 정렬 탭 클릭 시 실제 데이터 정렬:
    - `latest`: id 내림차순 (mock 기본 순서)
    - `views`: views 숫자 내림차순
    - `likes`: likes 내림차순
  - 프로젝트 카드 count를 필터링된 결과 길이로 업데이트
- 검증: `npx playwright test e2e/explore.spec.ts`

### Task 6 — EditorPage 실행 토글 + 저장 토스트 + 공유 모달 (4분)
- 파일: `frontend/src/pages/EditorPage/EditorPage.tsx`
- 구현:
  - `isRunning` state: "▶ 실행하기" / "⏹ 멈추기" 버튼 토글
  - 실행 중 상태바 "준비됨" → "▶ 실행 중..." + 스테이지에 시각적 표시

  - "🔗 공유하기" 클릭 → `<ShareModal />` open
  - `useToast` / `ShareModal` 연결
- 검증: `npx playwright test e2e/editor.spec.ts`

### Task 7 — PlayPage 오버레이 + 좋아요 토글 + 공유 모달 (4분)
